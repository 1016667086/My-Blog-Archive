### 计算机硬件系统

#### 计算机硬件组成

计算机硬件系统主要由**运算器、控制器、存储器、输入设备和输出设备**五大部件组成。运算器负责**算术运算**和**逻辑运算**；控制器是计算机的**指挥中心**，协调各部件工作；**存储器**用于存储数据和程序；输入设备如键盘、鼠标等负责将外部信息输入计算机；输出设备如显示器、打印机等将计算机处理结果输出。

#### 中央处理器（CPU）

1.CPU 的组成：CPU 由**运算器和控制器**组成，现代 CPU 还集成了高速缓存（Cache）。运算器中的算术逻辑单元（ALU）进行算术和逻辑运算，`累加器（ACC）`用于暂存运算结果。控制器包含程序计数器（PC），它存储下一条要执行指令的地址，指令寄存器（IR）存放当前正在执行的指令 。

1.CPU 的性能指标：

**主频**：即 CPU 的时钟频率，单位是 Hz，主频越高，CPU 运算速度越快。

**字长**：指 CPU 一次能处理的二进制数据的位数，字长越长，数据处理能力越强。例如 32 位 CPU 一次能处理 32 位二进制数据，64 位 CPU 则更强大。缓存的容量越大、速度越快、CPU访问数据的速度就越快，就减少CPU等待数据从内存读取的时间。

#### 存储系统

1.内存：内存分为**随机存取存储器（RAM）**和**只读存储器（ROM）**。RAM 用于临时存储正在运行的程序和数据，断电后数据丢失，常见的有**静态随机存储器（SRAM）**和**动态随机存储器（DRAM）**。SRAM 速度快但成本高、容量小，常用于 Cache；DRAM 成本低、容量大但速度相对较慢，是计算机的主存。ROM 用于存储固定不变的程序和数据，如 BIOS 程序，断电后数据不丢失。

外存：外存包括硬盘、光盘、U 盘等。硬盘是计算机中最常用的外存设备，其存储容量大、成本相对较低。光盘分为 CD、DVD、蓝光光盘等，根据读写特性又可分为只读光盘、一次写入光盘和可擦写光盘。U 盘则具有体积小、携带方便、读写速度较快等特点。

存储体系结构：计算机采用层次化的存储体系结构，从高速度、小容量的 Cache 到低速、大容量的外存，形成 **Cache - 主存 - 外存**三级存储结构。这种结构旨在解决存储系统中速度、容量和成本之间的矛盾，通过将经常访问的数据存放在高速的 **Cache** 和主存中，提高数据访问效率。

#### 总线系统

总线分类：

1. **数据总线**：用于在 CPU、内存和 I/O 设备之间传输数据，其宽度决定了一次能传输的数据量，如 32 位数据总线一次能传输 32 位数据。
2. **地址总线**：用于传输 CPU 访问内存或 I/O 设备的地址信息，地址总线的宽度决定了 CPU 的寻址能力。例如 32 位地址总线的寻址空间为 2^32 = 4GB。
3. **控制总线**：用于传输控制信号，协调各部件之间的操作，如读 / 写信号、中断请求信号等。

1.总线标准：常见的总线标准有 PCI（Peripheral Component Interconnect）、PCI - Express 等。PCI 是一种广泛应用的局部总线标准，用于连接计算机中的各种外部设备。PCI - Express 是新一代的总线标准，具有更高的数据传输速率，能够满足高速设备如显卡、固态硬盘等的需求。

#### 计算机软件系统

##### 操作系统

1.操作系统的功能：

* **进程管理**：负责进程的**创建、撤销、调度和同步**等。进程是程序的一次执行过程，操作系统通过进程管理合理分配 CPU 资源，实现多任务并发执行。例如在 **Windows** 系统中，用户可以同时打开多个应用程序，操作系统通过进程调度让这些程序交替使用 **CPU**资源。
* 存储管理：负责内存的**分配、回收和保护**等。操作系统为每个进程分配内存空间，防止进程之间的内存冲突。同时，通过虚拟内存技术，将一部分硬盘空间模拟成内存使用，扩大了程序可使用的内存空间。
* 设备管理：负责管理计算机的各种外部设备，如磁盘、打印机等。操作系统提供设备驱动程序，实现设备与计算机之间的通信和控制，使得用户能够方便地使用各种设备。
* 文件管理：负责文件的存储、检索、共享和保护等。操作系统以文件和目录的形式组织和管理存储在外存中的数据，用户可以通过文件名方便地访问和操作文件。
* 作业管理：负责作业的提交、调度和控制等。作业是用户在一次计算过程中要求计算机系统所做工作的集合，操作系统通过作业管理提高系统的资源利用率和用户的工作效率。

操作系统的类型：

* 批处理操作系统：用户将作业成批提交给计算机，计算机自动依次处理这些作业，期间用户不能干预。批处理操作系统的优点是系统资源利用率高，缺点是用户响应时间长。
* 分时操作系统：将 **CPU** 时间划分成时间片，轮流分配给多个用户进程使用，每个用户都感觉自己独占计算机。分时操作系统具有交互性强、响应及时等优点，常见的分时操作系统有UNIX、Linux 等。

  **实时操作系统**：能及时响应外部事件的请求，在规定的时间内完成对该事件的处理。实时操作系统主要用于工业控制、航空航天等对时间要求严格的领域，如 VxWorks、RT - Linux 等。

  网络操作系统：运行在网络服务器上，提供网络通信和网络资源共享功能的操作系统，如 Windows Server、Linux Server 等。

**分布式操作系统**：将多个分散的计算机通过网络连接起来，形成一个有机的整体，用户感觉就像使用一台计算机一样。分布式操作系统具有分布性、并行性、透明性等特点。
