3.(简答题18.0分)  
已知有以下程序代码:

```
void main()
int a,b,c;
if(a<1 && b>0)
c=2
else if(b<-1)
c=3;
else
c=4

```

画出这段代码的流程图,并分别采用语句图盖,判定覆盖、条件覆盖,条件组合覆盖和路径覆盖的方法设计测试用例。

（1）程序流程图

```
开始
↓
输入a, b
↓
a < 1 且 b > 0 ?
↓    ┌───────是───────┐
↓    ↓               ↓
否   c = 2           否
↓    ↓               ↓
b < -1 ?            是
↓    ┌───────是───────┘
↓    ↓
否   c = 3
↓    ↓
    c = 4
↓
结束

```

（2）测试用例设计  
① 语句覆盖  
用例 1：a=0, b=1（覆盖语句c=2）  
用例 2：a=2, b=-2（覆盖语句c=3）  
用例 3：a=2, b=0（覆盖语句c=4）

② 判定覆盖  
用例 1：a=0, b=1（判定 1 为真，判定 2 不执行）  
用例 2：a=2, b=-2（判定 1 为假，判定 2 为真）  
用例 3：a=2, b=0（判定 1 为假，判定 2 为假）

③ 条件覆盖  
用例 1：a=0, b=1（条件 a<1 为真，b>0 为真）  
用例 2：a=2, b=-1（条件 a<1 为假，b>0 为假，b<-1 为假）  
用例 3：a=1, b=-2（条件 b<-1 为真）

④ 条件组合覆盖  
组合 1：a<1 为真，b>0 为真 → 用例：a=0, b=1  
组合 2：a<1 为真，b>0 为假 → 用例：a=0, b=-1  
组合 3：a<1 为假，b>0 为真 → 用例：a=2, b=1  
组合 4：a<1 为假，b>0 为假 → 用例：a=2, b=-1  
组合 5：b<-1 为真 → 用例：a = 任意，b=-2  
组合 6：b<-1 为假 → 用例：a = 任意，b=0

⑤ 路径覆盖  
路径 1：a<1&&b>0 → c=2 → 用例：a=0, b=1  
路径 2：a<1&&b>0 为假 → b<-1 为真 → c=3 → 用例：a=2, b=-2  
路径 3：a<1&&b>0 为假 → b<-1 为假 → c=4 → 用例：a=2, b=0
